# 旧物志 (Relic Archive) 软件说明书

## 1. 软件概述

### 1.1 软件名称
旧物志 (Relic Archive)

### 1.2 软件版本
v1.0.0

### 1.3 软件定位
为恋物癖及收藏型用户打造的“旧物数字档案馆”，提供一个「情绪型数字收藏空间」。

### 1.4 核心价值
- 记录：完整记录旧物的细节和状态
- 分类：智能整理，降低用户收纳成本
- 情绪赋值：赋予物品故事感和生命力
- 数字重构：通过伪3D技术模拟物品存在感
- 私密收藏：本地加密，安全存储

## 2. 软件组成

### 2.1 系统架构
- **前端**：Flutter (Dart) - 跨平台移动应用
- **后端**：Python (FastAPI) - 提供API服务
- **AI**：OpenAI GPT-4o / Claude 3.5 (文本生成), YOLO/MobileNet (图像识别)
- **存储**：PostgreSQL, S3/OSS

### 2.3 核心模块
1. **相机模块**：多角度拍照、视频录制、AI辅助
2. **物品管理**：创建、编辑、分类、标签
3. **收藏馆**：创建主题收藏馆、隐私设置

5. **故事模式**：沉浸式浏览体验
6. **设置**：隐私管理、生物识别认证

## 3. 功能规格

### 3.1 多角度拍照记录系统
- **功能**：
  - 多角度拍摄引导（自动提示补拍缺失角度）
  - 局部细节拍摄（支持微距拍摄）
  - 视频短拍（最长10秒）
  - 拍摄备注（支持文字 + 语音情感记录）
- **AI 辅助**：
  - 自动识别物体边界
  - 自动光线校正
  - 自动去背景（可选）

### 3.2 自动分类系统
- **一级分类**：
  - 衣物、电子用品、家具、玩具、纸制品、生活用品、私密用品（支持隐藏）、自定义分类
- **AI 分类逻辑**：
  - 基于图像识别和物体特征
  - 学习用户历史分类习惯
- **标签系统**：
  - 物理属性：年代、材质、保存状态
  - 使用属性：使用频率
  - 情绪标签：安全感、怀旧、私密、欲望、放不下、纪念

### 3.3 AI 重构系统
- **自动生成旧物描述**：
  - 生成维度：外观描述、使用痕迹、推测年代、材质分析、情绪氛围
  - 风格选择：客观档案式、温柔叙述式、博物馆标签式、私密日记式
- **3D 重构**：
  - 伪3D浏览：通过多图拼接、视差动效、轻微旋转动画
  - 未来规划：基于 Gaussian Splatting 的高质量 3D 模型、AR 投影

### 3.4 收藏空间系统
- **功能**：
  - 创建收藏馆（设定主题）
  - 隐私设置（公开/私密）
  - 虚拟展览生成（瀑布流、网格、沉浸式）

### 3.5 隐私与心理安全机制
- **功能**：
  - 本地加密存储
  - FaceID/生物识别解锁
  - 伪装入口（可选）
  - 强隐私模式（默认不分享）

## 6. 使用方法

### 6.1 首次使用
1. 下载并安装应用
2. 打开应用，完成引导页
3. 注册或登录账号
4. 开始记录第一个旧物

### 6.2 记录旧物
1. 点击首页底部的相机图标
2. 按照引导拍摄多角度照片
3. 可选择录制视频（最长10秒）
4. 上传完成后，系统会自动生成AI描述
5. 编辑物品信息，添加标签和分类
6. 保存到收藏馆

### 6.3 浏览收藏
1. 点击首页的“我的收藏馆”
2. 选择收藏馆查看其中的物品
3. 点击物品进入详情页
4. 左右拖动查看伪3D效果
5. 阅读AI生成的故事

### 6.4 管理收藏馆
1. 进入“我的收藏馆”页面
2. 点击“创建收藏馆”按钮
3. 输入收藏馆名称和主题
4. 设置隐私权限
5. 添加物品到收藏馆

### 6.5 使用时间轴
1. 进入“时间轴”页面
2. 查看按时间排序的物品事件
3. 点击事件查看详情
4. 可播放音频备注
5. 可编辑或删除事件

### 6.6 隐私设置
1. 进入“设置”页面
2. 开启生物识别认证
3. 设置隐私模式
4. 管理收藏馆的隐私权限

## 7. 故障排除

### 7.1 常见问题
1. **相机无法启动**
   - 检查相机权限
   - 重启应用
   - 检查设备相机是否正常

2. **图片上传失败**
   - 检查网络连接
   - 尝试重新上传
   - 检查存储空间

3. **生物识别失败**
   - 确保设备支持生物识别
   - 检查生物识别设置
   - 尝试使用密码登录

4. **应用崩溃**
   - 重启应用
   - 检查应用版本
   - 联系客服
