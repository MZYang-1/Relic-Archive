# 旧物志 (Relic Archive)

> 为恋物癖及收藏型用户打造的“旧物数字档案馆”，一个「情绪型数字收藏空间」。

## 📖 项目简介
本项目旨在帮助用户记录、分类并赋予旧物以情绪价值。它不是一个二手交易平台，也不是断舍离工具，而是专注于物品记忆与情感连接的数字化空间。

## 🚀 核心功能 (MVP)
1.  **多角度拍照记录**：引导式拍摄，记录物品细节。
2.  **AI 智能分类**：基于图像识别自动分类。
3.  **情绪化 AI 描述**：生成具有“故事感”的物品描述。
4.  **私密收藏馆**：本地加密，安全存储。
5.  **伪 3D 浏览**：通过视差和多图拼接模拟物体存在感。

## 🛠 技术栈
- **移动端**: Flutter (Dart) - 位于 `app/` 目录
- **后端**: Python (FastAPI) - 位于 `backend/` 目录
- **AI**: OpenAI GPT-4o / Claude 3.5 (文本生成), YOLO/MobileNet (图像识别)。
- **存储**: PostgreSQL, S3/OSS。
